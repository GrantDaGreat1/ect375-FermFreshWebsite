# Ferm Fresh Website 2.0

Ferm Fresh Website 2.0 is a modern web application built to present the Ferm Fresh brand with a clean, responsive experience and a scalable backend foundation.

## Vibe Coding Stack

- Next.js
- Supabase
- Antigravity

## How to Run

1. Install dependencies.
2. Start the development server:

	npm run dev

3. Open the local development URL shown in your terminal.
